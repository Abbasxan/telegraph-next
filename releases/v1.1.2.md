## 🌟 Overview
Version 1.1.2 marks the official revival and rebranding of the project to **`telegraph-next`**. We migrated the core logic to modern asynchronous frameworks and fixed critical bugs that plagued the original library.

---

## 🛠️ Key Changes
* **Rebranded:** Officially launched as `telegraph-next`.
* **Async Core:** Fully migrated dependencies to `aiohttp` and `pydantic`.
* **Fixed Memory Leaks:** Implemented strict `try-finally` blocks in `upload_file()` to properly release file streams.
* **Fixed Parsers:** Resolved `AttributeError` during HTML sanitization and fixed YouTube iframe embedding logic.
* **Documentation:** Completely rewritten `README.md` with a premium landing-page design and clear examples.
* **CI/CD:** Added automated GitHub Actions workflows for PyPI publishing.
